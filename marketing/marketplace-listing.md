# VS Code Marketplace Draft

## Name

Prompt Privacy Shield

## Tagline

Sanitize code and logs before sending them to AI.

## Short Description

Remove common secrets, local paths, private IPs, emails, and tokens from prompts
before pasting them into AI coding tools.

## Long Description

Prompt Privacy Shield helps developers use AI coding tools without accidentally
pasting sensitive context.

Select code, logs, stack traces, or configuration text in VS Code, then run a
sanitize command. The extension redacts common risky patterns and copies the
safe version to your clipboard, ready for ChatGPT, Codex, Cursor, Claude, or a
bug report.

## Commands

- `Prompt Privacy Shield: Sanitize Selection`
- `Prompt Privacy Shield: Sanitize Clipboard`

## Keywords

AI privacy, prompt security, sanitize, redact, secrets, API key, token, ChatGPT,
Codex, Cursor, Claude, VS Code
