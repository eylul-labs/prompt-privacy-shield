# Product Brief

## Product

Prompt Privacy Shield

## Positioning

Sanitize code, logs, and prompts before sending them to AI tools.

## Target User

Developers who use ChatGPT, Codex, Cursor, Claude, or similar AI coding tools
and frequently paste logs, source snippets, stack traces, and configuration
values into prompts.

## Pain

Developers want AI help quickly, but pasted context can include secrets,
private infrastructure details, emails, local paths, or internal URLs.

## MVP

- VS Code command to sanitize selected text.
- VS Code command to sanitize clipboard text.
- Clipboard output with sanitized text.
- Markdown report showing detected pattern categories.
- Core sanitizer tests for high-signal sensitive patterns.
